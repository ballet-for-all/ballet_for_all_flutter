# 모두의 발레 앱

## 개요

모두의 발레 서비스를 위한 플러터 앱입니다.

## 개발 환경

- Flutter 3.7.0 with FVM

### 개발 환경 구축 팁

[FVM](https://fvm.app/) 설치 후 아래 명령어를 실행합니다.

```bash
  $ fvm install 3.7.0
  $ fvm flutter pub get
```

사용하는 IDE에 따라 fvm 을 인식하도록 설정합니다.([#](https://fvm.app/docs/getting_started/configuration#ide))

## 실행 방법

### 의존성 설치

```bash
  $ fvm flutter pub get
```

### 실행

```bash
  $ fvm flutter run
```
